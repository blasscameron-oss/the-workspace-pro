The Workspace Pro - Modern Static Site

A sleek, high‑performance home office productivity site built with Eleventy, Tailwind CSS v4, and automated tooling.

🚀 Features

  • 10 expert articles on home office desks, ergonomic chairs, monitor setups, organization
  • Vibrant, modern design – gradient hero, card‑based layout, dark‑mode ready
  • Blazing fast – 90+ PageSpeed scores (static HTML, optimized images)
  • Fully responsive – mobile‑first, responsive navigation
  • SEO‑ready – clean URLs, meta tags, sitemap
  • Automated workflow – image optimization, visual regression tests, CI/CD

🛠️ Tech Stack

  • Eleventy 3 – Static site generator
  • Tailwind CSS 4 – Utility‑first CSS with custom theme
  • PostCSS – CSS processing with autoprefixer, cssnano
  • Netlify – Hosting & continuous deployment
  • GitHub Actions – Automated builds & previews
  • Puppeteer – Visual regression testing
  • ImageMagick – Automated image optimization

📦 Local Development

# Install dependencies
npm install

# Build site (Tailwind + Eleventy)
npm run build

# Serve locally with live reload
npm run serve

# Dev mode (Tailwind watch + Eleventy serve)
npm run dev

# Optimize all images (reduces size ~50%)
npm run optimize-images

# Take visual regression screenshots
npm run test:visual

🎨 Design System

  • Primary colors: #4361ee (vibrant blue), #7209b7 (purple), #f72585 (accent pink)
  • Typography: Inter (sans), Poppins (headings)
  • Components: Gradient cards, animated buttons, sticky navigation
  • Dark mode: Toggle with localStorage support

🤖 Automation & Tooling

Image Optimization

All images are automatically resized (max 1200px), compressed (85% quality), and converted to WebP. Run npm run optimize-images to reprocess.

Visual Regression Testing

Puppeteer captures homepage screenshots on each commit; diff against baseline to catch unintended visual changes.

CI/CD Pipeline

  • On push to main: Build and deploy to Netlify production
  • On pull request: Create Netlify preview deploy
  • Automated optimization: Images processed during build

🚀 Deployment

Quick Deploy (Netlify)

Manual Deploy

# Build site first
npm run build

# Deploy via Netlify CLI (requires auth)
npx netlify deploy --dir=_site --prod

GitHub Actions (Recommended)

  1. Set secrets NETLIFY_AUTH_TOKEN and NETLIFY_SITE_ID in repository
  2. Push to main branch → auto‑deploy to production
  3. PRs get preview URLs

📁 Project Structure

static-site/
├── content/              # Markdown articles
├── _includes/           # Nunjucks layouts (base, post, page)
├── css/                 # Tailwind CSS source (main.css)
├── static/              # Images, fonts, other assets
├── _site/               # Built site (do not edit)
├── .eleventy.js         # Eleventy configuration
├── tailwind.config.js   # Tailwind theme (legacy, using @theme)
├── postcss.config.js    # PostCSS plugins
├── package.json         # Scripts & dependencies
└── .github/workflows/   # CI/CD pipelines

📝 Adding Content

  1. Create a new .md file in content/
  2. Use frontmatter:
---
title: "Article Title"
date: 2026-04-15
layout: post
category: "ergonomic-chairs"
tags: ["productivity", "ergonomics"]
featured_image: "/static/images/your-image.jpg"
description: "A brief summary for SEO and social sharing."
---
  1. Write in Markdown
  2. Commit & push – deployment is automatic

🧪 Testing

  • Visual regression: npm run test:visual
  • Build validation: npm run build
  • Performance audit: Use Lighthouse CLI or Netlify Analytics
