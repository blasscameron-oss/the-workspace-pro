🚀 Deploy The Workspace Pro to Netlify

Your static site is ready for deployment. Choose Option 1 for the fastest path (drag‑and‑drop).

📦 What's Ready

  • Site files: _site/ folder (16 HTML files + 53 images)
  • Build system: Eleventy configured (package.json, .eleventy.js)
  • Netlify config: netlify.toml (redirects, headers, build command)
  • Zip archive: static-site-deploy.zip (ready for drag‑and‑drop)

Option 1: Drag & Drop (5 minutes, no CLI)

  1. Go to app.netlify.com
  2. Log in with GitHub, GitLab, or email
  3. Drag the _site folder (or static-site-deploy.zip) onto the Netlify dashboard
  4. Wait for automatic deployment (≈30 seconds)
  5. Get your URL: https://[random-name].netlify.app

✅ Done! Your site is now live on Netlify’s global CDN.

Option 2: Netlify CLI (Developer workflow)

# Install Netlify CLI globally
npm install -g netlify-cli

# Login to Netlify
netlify login

# Deploy from the static-site directory
cd /home/cameron/.openclaw/workspace/static-site
netlify deploy --prod --dir=_site

Follow prompts to create a new site or link to existing.

Option 3: Connect GitHub Repository

  1. Push to GitHub:

    cd /home/cameron/.openclaw/workspace/static-site
git init
git add .
git commit -m "Initial static site"
# Create repo on GitHub, then:
git remote add origin https://github.com/yourusername/theworkspacepro.git
git push -u origin main

  2. In Netlify: Click "New site from Git", select your repository

  3. Build settings: Netlify auto‑detects Eleventy (npm run build)

  4. Deploy: Click "Deploy site"

🌐 Connect Custom Domain (theworkspacepro.com)

After deployment:

  1. In Netlify dashboard, go to Site settings → Domain management
  2. Click "Add custom domain"
  3. Enter theworkspacepro.com
  4. Choose "Add domain" (Netlify will suggest DNS records)

DNS Update (Namecheap)

  1. Log in to Namecheap

  2. Go to Domain List → Manage for theworkspacepro.com

  3. Update nameservers to Netlify’s:

    dns1.p08.nsone.net
dns2.p08.nsone.net
dns3.p08.nsone.net
dns4.p08.nsone.net

    Or add A records (if keeping Namecheap DNS):

    • @ → 75.2.60.5
    • www → 75.2.60.5

  4. Propagation: 5‑60 minutes (global DNS update)
