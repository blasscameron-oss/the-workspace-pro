Affiliate Link Setup

Your site now includes a product‑recommendation system with affiliate‑link support.

How It Works

  1. Product database: _data/products.json contains all recommended products.
  2. Automatic display: Products linked to articles via articleSlugs appear in a "Recommended Products" section at the bottom of each article.
  3. Affiliate URLs: Each product has an affiliateUrl field you can update with your own affiliate links.

Setting Up Affiliate Programs

1. Amazon Associates

  • Sign up at affiliate-program.amazon.com
  • Once approved, use the Amazon SiteStripe to generate links, or manually add your tag:
    • Example: https://www.amazon.com/dp/B07N4Q5H6Z?tag=yourname-20

2. Other Retailers

  • Staples: Staples Affiliate Program
  • Dell: Dell Partner Program
  • Logitech: Logitech Partner Program
  • IKEA: IKEA does not have a traditional affiliate program; consider linking to product pages directly.

3. Update _data/products.json

Replace the placeholder affiliateUrl values with your actual affiliate links.

Example:

{
  "affiliateUrl": "https://www.amazon.com/dp/B07N4Q5H6Z?tag=workspacepro-20"
}

Legal Requirements

FTC Disclosure: The site already includes:

  • Footer disclosure ("Some links on this site are affiliate links...")
  • Per‑section note in the "Recommended Products" block

Ensure compliance:

  • Disclosures must be clear and conspicuous
  • Do not make false claims about products
  • Monitor affiliate program terms for each retailer

Adding New Products

  1. Add a new object to _data/products.json
  2. Include the articleSlugs array to link to specific articles
  3. Run npm run build to regenerate the site

Tracking Performance

Consider adding Google Analytics or a dedicated affiliate‑tracking platform to monitor which links convert best.
